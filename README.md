1-HTML:

        During the course of this project i used the following HTML tags:

    <!DOCTYPE html>:it's a tag used to tell the web brower that it is currently reading a html document and should render it as a webpage.

    <html>: it's the root element that contain the entire page

    <head>:it's a tag used as a container for metadata

    <link>:it's used to link external ressources to the html page, most commonly , it's used to link css files

    <title>: used to set the pages' title

    <body>:it is used to highlight the body of the webpage that's the part that the user will actually see

    <h1>: it is a tag used in html to highlight a heading;

    <a>: it's a tag used to create hyperlinks in html

    <p>: it is used to create paragraphs

    <div>: it's used to group and organize html element

    <footer>: used to set the bottom section of a html page.


        i also made use some attributes like:

    id: it's used to give a unique name to a html element

    class: it's used to groupe element to style or script them together

    href: it defines the destination url of a link
